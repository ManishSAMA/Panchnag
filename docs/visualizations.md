# Visualizations

The `visualize.py` module provides visual feedback and debugging tools for Panchang data generated by the app. It uses Matplotlib and Pandas to turn astronomical numeric data into charts and console-readable output.

All commands below assume you have already generated a CSV file using the CLI or web app.

## Planetary Motion Chart

Plots sidereal longitudes for all 9 planets across the generated timeframe. Useful for spotting retrograde periods, sign transitions, and overall orbital patterns.

```bash
python visualize.py --file panchang_2025_2025.csv --plot planets --year 2025
```

Filter to specific planets with `--planets`:

```bash
python visualize.py --file panchang_2025_2025.csv --plot planets --year 2025 --planets Sun Moon Mars
```

## Tithi Frequency Chart

Renders a bar chart counting the total occurrences of each Tithi across the dataset. Useful for verifying that lunar day distribution is reasonable over long ranges.

```bash
python visualize.py --file panchang_2025_2025.csv --plot tithis --year 2025
```

## Panchang Element Heatmap

Renders a month-by-month calendar heatmap for a chosen Panchang element. Useful for spotting seasonal patterns in Nakshatra, Yoga, or Karana distribution.

```bash
python visualize.py --file panchang_2025_2025.csv --plot calendar --element Nakshatra --year 2025
```

Valid elements: `Tithi`, `Nakshatra`, `Yoga`, `Karana`

## CSV Diff and QA Comparison

Merges two CSV outputs on their `Date` column and reports numeric discrepancies in planetary positions. Useful when switching ayanamsas or verifying that a code change did not alter output values unintentionally.

```bash
python visualize.py --file panchang_generated.csv --ref panchang_reference.csv --compare
```

## Console Debug Dump

Dumps all planetary positions and the five Panchang elements for a single date directly to the terminal. Useful when debugging edge cases without generating a full CSV.

```bash
python visualize.py --debug --date 2025-01-14 --lat 26.9124 --lon 75.7873
```

This prints all 9 planetary positions in decimal and DMS form, along with Tithi, Nakshatra, Yoga, Karana, and Vara for the given date and coordinates.
