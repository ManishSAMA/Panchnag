# Visualizations Guide

The `visualize.py` module gives you visual feedback on the planetary data generated by the primary script. Utilizing Matplotlib and Pandas, it translates complex astronomical numeric data into easy-to-read charts.

## Setup Requirements
Visualization relies on data analysis and rendering packages. Ensure they are installed:
```bash
pip install pandas matplotlib
```

---

## 📈 Planetary Motion Plotting
Plot the sidereal longitudes of planets across your generated timeframe. The chart visually highlights periods when planets enter specific constellations (Rashis).

**Usage:**
```bash
python visualize.py --file panchang_2025_2025.csv --plot planets --year 2025
```
You can also filter specific planets by adding: `--planets Sun Moon Mars`. The script dynamically graphs their exact orbital movement patterns.

---

## 📊 Tithi Frequency Chart
Vedic Tithis run on lunar day definitions, meaning their lengths continuously vary. This distribution graph renders a bar chart counting the total occurrences of each respective Tithi type across the entire dataset.

**Usage:**
```bash
python visualize.py --file panchang_2025_2025.csv --plot tithis --year 2025
```

---

## 🗓️ Heatmap Calendar
A robust month-by-month layout portraying consecutive index triggers for any major Panchang Element (Nakshatra, Yoga, Karana). It isolates anomalies and quickly provides seasonal visual pacing.

**Usage:**
```bash
python visualize.py --file panchang_2025_2025.csv --plot calendar --element Nakshatra --year 2025
```
*(Valid elements: `Tithi`, `Nakshatra`, `Yoga`, `Karana`)*

---

## 🛠️ File Comparison (QA/Diff Tool)
Ideal for validation if switching Ayanamsas. Merges two output datasets using their `Date` columns and strictly measures numeric discrepancies on planetary positioning decimals.

**Usage:**
```bash
python visualize.py --file panchang_generated.csv --ref panchang_reference.csv --compare
```

---

## 🐞 Console Debug Dumps
Forces an absolute bypass of CSV saving and directly parses a single date inside the terminal console. Highly useful for debugging edge cases on arbitrary chronological anomalies.

**Usage:**
```bash
python visualize.py --debug --date 2025-01-14 --lat 26.9124 --lon 75.7873
```
This isolates all 9 planetary position decimals alongside the 5 major Panchang configurations into an indexed, formatted list within the CLI window, preventing the need to dig through massive CSV spreadsheets during active algorithmic development.
